# 🔱 Aegis System Framework

A high-performance Discord administrative framework designed for advanced server management and automated maintenance. The Aegis System features a dual-layer interface: a public Slash Command suite for community interaction and a secure, invisible Master Console for authorized administrators.

## 🚀 Key Features

* **Dual-Layer Interface**: Seamlessly switch between public `/` commands and hidden `?` triggers.
* **Zero-Footprint Stealth**: Master commands are automatically purged from text channels to maintain a clean environment.
* **Infrastructure Management**: Rapid channel and role deployment/cleanup tools.
* **Security First**: Credential management via environment variables to prevent token leaks.
